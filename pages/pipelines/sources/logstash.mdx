### Configure Logstash Pipeline

Add a [TCP output](https://www.elastic.co/guide/en/logstash/current/plugins-outputs-tcp.html) to your Logstash pipeline, and configure the host and port to point to the Cardinal Collector's Service URL. 

Make sure to set the `codec` to `json_lines`

```javascript copy
output {
  tcp {
    host => "mycollector.cardinalhq.svc.cluster.local"
    port => 2255
    codec => json_lines
  }
}
```
