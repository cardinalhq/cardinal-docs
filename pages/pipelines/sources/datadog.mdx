### Configure Datadog Agent

Update your Datadog Agent configuration to point to the Cardinal Collector's Service URL by setting the `dd_url`, `logs_dd_url`, and `apm_dd_url` properties.

```yaml copy
datadog:
  dd_url: "http://mycollector.cardinalhq.svc.cluster.local:8126"

  customAgentConfig:
    apm_config:
      apm_dd_url: "http://mycollector.cardinalhq.svc.cluster.local:8126"

    logs_config:
      logs_dd_url: "mycollector.cardinalhq.svc.cluster.local:8126"
      use_http: true
      logs_no_ssl: true
```
