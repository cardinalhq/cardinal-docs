# Rules

Cardinal Collector Pipelines support a variety of rule types that can be applied to telemetry data, including:

- [Add Tags](./rules/add-tags)
- [Drop Tags](./rules/drop-tags)
- [Filter](./rules/filter)
- [Sample](./rules/sample)
- [Extract Metric](./rules/extract-metric)
- [Drop/Mask PII](./rules/pii)
- [Custom OTTL](./rules/custom)

Refer to the documentation for each Rule type for more information on how to configure it.
