# Sources

Cardinal Collector Pipelines support a variety of telemetry sources, including:

- [Datadog](./sources/datadog)
- [Logstash](./sources/logstash)
- [OpenTelemetry (OTLP)](./sources/opentelemetry)
- [Prometheus](./sources/prometheus)

Refer to the documentation for each Source for more information on how to configure it.

If you're looking for a Source that isn't listed here, please [email us](mailto:support@cardinalhq.io) and let us know!
