# Connectors

Connectors allow the Cardinal Agent Builder to query and interact with your data sources.

## Databases

| Connector | Description |
|-----------|-------------|
| [**PostgreSQL**](/connectors/postgres) | PostgreSQL databases (Amazon RDS, Google Cloud SQL, self-hosted) |
| **BigQuery** | Google BigQuery data warehouse |
| **Athena** | AWS Athena serverless SQL queries |
| **ClickHouse** | ClickHouse OLAP database |
| **LakeRunner** | CardinalHQ's OSS data lake |

## Observability

| Connector | Description |
|-----------|-------------|
| **CloudWatch** | AWS CloudWatch Logs and Metrics |
| **Datadog** | Datadog logs, metrics and traces |
| **Grafana** |  Loki, Mimir and Tempo |
| **Prometheus** | Prometheus metrics |
| **Thanos** | Thanos metrics |
| **GCP Monitoring** | Google Cloud Monitoring metrics and logs |

## Integrations

| Connector | Description |
|-----------|-------------|
| **GitHub** | GitHub repositories and releases |
| **Slack** | Slack notifications and messaging |
| **Lambda** | AWS Lambda function invocation |
