# Scripts

![Scripts](/Script.gif)

Scripts give you deterministic control over an agent’s behavior so critical actions run every time, predictably, instead of being left to the LLM’s discretion.

Generate scripts as you build agents, turning your workflow into repeatable, reliable actions.

## Key Benefits

- Extend Cardinal's capabilities with custom logic
- Automate complex multi-step operations
- Integrate with custom systems and tools
- Create reusable automation templates
- Handle edge cases and specialized scenarios
