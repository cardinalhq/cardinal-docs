Connect Cardinal Agent Builder to Prometheus for monitoring and analyzing metrics.

## Overview

Prometheus is an open-source monitoring and alerting system that collects and stores time-series metrics. With the Prometheus connector, your AI agents can:

- **Query Prometheus metrics** using PromQL
- **Analyze time-series data** with aggregations and functions
- **Monitor application performance** and infrastructure health
- **Track metric trends** over time
- **Identify anomalies** and performance issues

## Connection Details

| Field | Description |
|-------|-------------|
| **Display Name** | Custom name for this Prometheus connection |
| **Prometheus URL** | Base URL of your Prometheus server |

## Prerequisites

- Prometheus server deployed and running
- Base URL for your Prometheus server (e.g., `http://localhost:9090`)
- Network access to the Prometheus server
- Metrics data available in Prometheus
