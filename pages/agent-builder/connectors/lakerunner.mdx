Connect Cardinal Agent Builder to Cardinal HQ's open-source data lake.

## Overview

Lakerunner is CardinalHQ's open-source data lake solution that allows you to query and analyze structured data. With the Lakerunner connector, your AI agents can:

- **Query data** in your Lakerunner instance
- **Access structured tables** with schema discovery
- **Analyze data** with natural language queries
- **Integrate with your data lake** seamlessly
- **Manage data sources** through a unified interface

## Connection Details

| Field                 | Description                                    |
| --------------------- | ---------------------------------------------- |
| **Display Name**      | Custom name for this Lakerunner connection     |
| **API URL**           | URL of your Lakerunner API endpoint            |
| **API Key**           | API key for Lakerunner authentication          |
| **Share credentials** | (Optional) Share credentials with organization |

## Prerequisites

- Lakerunner instance deployed and running
- API URL for your Lakerunner instance
- Valid API key for authentication
- Access to data tables in Lakerunner

## Setup Wizard

### Step 1: Connection Configuration

1. In Cardinal Agent Builder, go to **Connectors** and click **Browse Connectors**
2. Select **Lakerunner**
3. Enter a **Display Name** for this connection (e.g., "Lakerunner Prod", "Lakerunner Dev")

### Step 2: Lakerunner Configuration

1. Enter your **API URL**
   - Example: `https://app.cardinalhq.io`
   - The URL of your Lakerunner API endpoint
2. Enter your **API Key**
   - Your authentication key for accessing Lakerunner

### Step 3: Credentials Storage (Optional)

1. Check **Share credentials with organization** to securely share with your organization
   - When disabled: credentials are stored locally on this device
   - When enabled: credentials are securely shared with your organization members

### Step 4: Save

1. Click **Save** to complete the configuration
2. Cardinal Agent Builder will verify connectivity to your Lakerunner instance

Your AI agent can now query your Lakerunner data lake.
