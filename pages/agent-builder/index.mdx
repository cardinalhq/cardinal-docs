# Cardinal Agent Builder

Cardinal Agent Builder is a desktop application that enables you to build, configure, and deploy custom AI agents. These agents can autonomously monitor your infrastructure, answer questions about your systems, and take action based on deployment events.

[Understand the assets and product features →](/agent-builder/product)

## What are AI Agents?

AI agents are autonomous assistants that can:
- Monitor your cloud infrastructure and deployments
- Query logs, metrics, and traces across multiple platforms
- Correlate deployment events with application behavior
- Provide insights and answer questions about your systems
- Execute workflows based on triggers and conditions

## Available Agents

### Release Agent

The Release Agent automatically tracks deployments across your infrastructure and correlates them with GitHub releases. It can:
- Monitor CloudRun and ECS deployments in real-time
- Correlate deployments with GitHub release notes
- Provide deployment history and insights
- Alert on deployment changes

[Get started with the Release Agent →](/agent-builder/release-agent)

## Getting Started

1. **Download Cardinal Agent Builder** from [cardinalhq.io](https://cardinalhq.io)
2. **Install** the application on your desktop (macOS, Windows, or Linux)
3. **Configure** your first agent following the guides in this section
4. **Connect** your infrastructure using supported connectors

## Prerequisites

- A Cardinal account with API key
- GitHub integration configured (for Release Agent)
- Access to your cloud platform (GCP, AWS, etc.)
