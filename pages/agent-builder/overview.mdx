# Cardinal Agent Builder

Cardinal Agent Builder is a [desktop application](http://www.cardinalhq.io) that enables you to build, configure, and deploy custom AI agents. These agents can autonomously monitor your infrastructure, answer questions about your systems, and take action based on deployment events.

[Product Features](/agent-builder/product)

## Agents Showcase

- [Datadog Release Agent](/agent-builder/agents/datadog-release-agent): An agent that monitors Datadog release notes and notifies you of relevant updates.
