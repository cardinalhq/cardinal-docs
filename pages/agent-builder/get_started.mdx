# Get Started

1. [Sign up](https://app.cardinalhq.io/signup) for a free Cardinal account.

2. On first login, you will be prompted to create and link a new GitHub repository that will be the shared workspace for your team's Agents and their assets.

![Create GitHub Repository](/signup/github-steps.png)

3. Click **Create New Repository** to create a new, empty repository in GitHub â€“ this can be a private repository if you like.

4. Next, click **Install GitHub App** to install the Cardinal GitHub App. On the install screen, select **only** the new, empty repository you created in the previous step.

![Install GitHub App](/signup/github-app-install.png)

5. Upon successful installation, you will be redirected to the Cardinal web UI. Make sure that you see the **"Repository connected"** message before proceeding.

![GitHub Connected](/signup/github-connected.png)

6. You and your team can now start importing, customizing, and deploying Agents using the Agent Builder desktop app. Click **Go to Download Page** to download and install the app on macOS, Windows, or Linux.

7. Learn more [about the product,](/agent-builder/product) or how to [import one of Cardinal's pre-built Agents](/agent-builder/agents/datadog-release-agent) for common use cases that can be customized to fit your needs.
