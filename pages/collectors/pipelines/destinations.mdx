# Destinations

Cardinal Collector Pipelines support a variety of destinations, including:

- [Datadog](./destinations/datadog)
- [OpenTelemetry (OTLP)](./destinations/opentelemetry)
- [Prometheus](./destinations/prometheus)

Refer to the documentation for each Destination for more information on how to configure it.

If you're looking for a Destination that isn't listed here, please [email us](mailto:support@cardinalhq.io) and let us know!
